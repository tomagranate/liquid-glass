# Demo media

Screenshots and recordings referenced by the top-level [README](../README.md).
Capture them from the live demo (`npm run dev` → http://localhost:5180).

- `hero.gif` — the showcase stage with the glass controls in motion (drag a
  slider, toggle a switch, tap the QR code). ~6–10s loop, ≤ 1280px wide,
  optimized.
- `components.png` — a clean still of the button / toggle / switch / slider row.

Capture tips: record at 2× / retina, then downscale; keep GIFs under ~3 MB (or
record an MP4/WebM and reference it instead — GitHub renders committed videos
inline in Markdown). Commit the files into this `assets/` directory with the
names the README expects.
